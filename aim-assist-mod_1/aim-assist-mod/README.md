# 🎯 Aim Assist — Fabric Mod (Minecraft 1.21.1)

Плавное автонаведение на ближайшего врага при атаке.

---

## ✨ Возможности

- **Мягкое сведение** — камера плавно тянется к ближайшему врагу, не «прыгает»
- **Работает только при зажатой ЛКМ** — не мешает обычному движению камеры
- **Клавиша V** — включить / выключить в любой момент (сообщение в HUD)
- Без миксинов — только через события Fabric API

---

## 📦 Установка

1. Установи [Fabric Loader](https://fabricmc.net/use/) для **Minecraft 1.21.1**
2. Скачай [Fabric API](https://modrinth.com/mod/fabric-api) и положи в папку `mods/`
3. Скомпилируй мод (`./gradlew build`) → возьми файл из `build/libs/`
4. Положи `.jar` в папку `mods/`
5. Запусти игру 🎮

---

## ⚙️ Настройки (редактировать вручную)

В файле `AimAssistMod.java`:

| Константа        | По умолчанию | Описание                                              |
|------------------|-------------|-------------------------------------------------------|
| `SEARCH_RADIUS`  | `6.0`       | Радиус поиска целей в блоках                          |
| `MAX_ANGLE`      | `30.0`      | Максимальный угол от прицела для захвата цели (°)     |
| `SMOOTHING`      | `0.08`      | Плавность: 0.01 = очень медленно, 0.3 = резко         |

---

## 🛠 Сборка

```bash
./gradlew build
```
Готовый `.jar` будет в `build/libs/aim-assist-1.0.0.jar`

---
